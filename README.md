## Maven Electronics E-Commerce Sales Analysis

### Overview

This project analyzes sales performance, customer behavior, product trends, and regional growth for Maven Electronics using an interactive business intelligence dashboard.

The goal is to transform raw transactional data into actionable insights that support strategic decision-making in an e-commerce environment.

### Tools Used

* Power BI
* Power Query
* DAX

### Key Analysis Areas

* **Sales Performance:** Revenue, profit, quantity sold, and yearly trends
* **Product Analysis:** Top and bottom performing products and category profitability
* **Customer Insights:** New vs returning customers, gender distribution, and age segments
* **Regional Analysis:** Revenue distribution across countries and store performance

### Key Insights

* Revenue and profit show consistent growth across most years.
* A small number of products drive a large share of total revenue.
* Returning customers contribute significantly to overall sales.
* Certain regions and stores outperform others in revenue generation.
